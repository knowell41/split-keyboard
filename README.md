# Dactyl Manuform 4x6 — Quick Reference Guide

Welcome to your first split keyboard! This guide covers your specific keymap layout to help you get comfortable. Take it slow — muscle memory will build over time.

---

## Your Keyboard at a Glance

Your Dactyl Manuform has **3 rows × 6 columns + 5 thumb keys on each side** (46 keys total) across **4 layers**. The thumb keys are the heart of this layout — they let you access every layer and symbol without moving your fingers off the home row.

---

## Layer 0 — Default (QWERTY)

This is your home base. Standard QWERTY with a few things to note.

```

├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│ TAB  │  A  │  S  │  D  │  F  │  G  │ │  H  │  J  │  K  │  L  │  ;  │  '   │
```

**Home row:** Place your fingers on `A S D F` (left) and `J K L ;` (right), just like a regular keyboard.

**Key things to remember:**
- Backspace is top-right (where you'd expect it)
- Both Shift keys are on the outer edges of the bottom row
- Brackets `[` and `]` are on the right thumb area — handy for coding

## How Layers Work

Your keyboard uses **layer-tap** keys on the thumbs. This means one key does two things depending on how you press it.

| Key | Tap (quick press) | Hold (press and hold) |
|-----|-------------------|-----------------------|
| `LWR/SPC` | Space | Activates Layer 1 (Lower) |
| `RSE/ENT` | Enter | Activates Layer 2 (Raise) |
| `FN` | Nothing | Activates Layer 3 (Bluetooth) |

The tapping threshold is **200ms**. If you press and release within 200ms, you get the tap action. Hold longer and you get the layer.

**Golden rule:** Hold a thumb key → press the key you need on that layer → release everything.

## Layer 1 — Lower (Hold LWR/SPC)

Hold your left thumb on `LWR/SPC` to access this layer. This gives you **function keys, navigation, and arrows**.

```
 LEFT HAND                              RIGHT HAND
┌──────┬─────┬─────┬─────┬─────┬─────┐ ┌─────┬─────┬─────┬─────┬─────┬──────┐
│  ~   │ F1  │ F2  │ F3  │ F4  │ F5  │ │ F6  │ F7  │ F8  │ F9  │ F10 │ DEL  │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │ F11 │ F12 │     │     │     │ │HOME │PgDn │PgUp │ END │     │  =   │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │     │     │     │     │     │ │  ←  │  ↓  │  ↑  │  →  │     │      │
└──────┴─────┴─────┴─────┴─────┴─────┘ └─────┴─────┴─────┴─────┴─────┴──────┘
```

**Common uses:**
- **Arrow keys** are on the right hand: `N` = Left, `M` = Down, `,` = Up, `.` = Right (like HJKL shifted one column over)
- **Delete** is top-right (same position as Backspace on Layer 0)
- **Home / End** and **PgUp / PgDn** for text navigation
- **F1–F12** across the top and second rows

## Layer 2 — Raise (Hold RSE/ENT)

Hold your right thumb on `RSE/ENT` to access this layer. This gives you **symbols and special characters**.

```
 LEFT HAND                              RIGHT HAND
┌──────┬─────┬─────┬─────┬─────┬─────┐ ┌─────┬─────┬─────┬─────┬─────┬──────┐
│  ~   │  !  │  @  │  #  │  $  │  %  │ │  ^  │  &  │  *  │  (  │  )  │ BSPC │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │     │     │     │     │     │ │  -  │  =  │  [  │  ]  │  \  │  `   │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │     │     │     │     │     │ │  _  │  +  │  {  │  }  │  |  │  ~   │
└──────┴─────┴─────┴─────┴─────┴─────┘ └─────┴─────┴─────┴─────┴─────┴──────┘
```

**Common uses:**
- **Top row** gives you the shifted number symbols: `! @ # $ % ^ & * ( )`
- **Middle row, right hand** gives you `-  =  [  ]  \` and backtick
- **Bottom row, right hand** gives you the shifted versions: `_  +  {  }  |` and tilde
- Great for coding — brackets, braces, pipes, and operators are all here

## Layer 3 — FN / Bluetooth (Hold FN)

Hold the left-most thumb key `FN` to access Bluetooth controls.

```
 LEFT HAND                              RIGHT HAND
┌──────┬─────┬─────┬─────┬─────┬─────┐ ┌─────┬─────┬─────┬─────┬─────┬──────┐
│BT CLR│ BT1 │ BT2 │ BT3 │ BT4 │ BT5 │ │     │     │     │     │     │      │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │     │     │     │     │     │ │     │     │     │     │     │      │
├──────┼─────┼─────┼─────┼─────┼─────┤ ├─────┼─────┼─────┼─────┼─────┼──────┤
│      │     │     │     │     │     │ │     │     │     │     │     │      │
└──────┴─────┴─────┴─────┴─────┴─────┘ └─────┴─────┴─────┴─────┴─────┴──────┘
```

| Action | Keys to press |
|--------|---------------|
| Pair to device slot 1 | Hold `FN` + tap `Q` |
| Pair to device slot 2 | Hold `FN` + tap `W` |
| Pair to device slot 3 | Hold `FN` + tap `E` |
| Pair to device slot 4 | Hold `FN` + tap `R` |
| Pair to device slot 5 | Hold `FN` + tap `T` |
| Clear current BT profile | Hold `FN` + tap `ESC` |

**Note:** Only the **left half** of your split keyboard connects to your computer via Bluetooth. The right half talks to the left half wirelessly.

## Everyday Cheat Sheet

Here are the most common things you'll need to do, and exactly which keys to press.

### Typing Essentials

| Action | How |
|--------|-----|
| Space | Tap `LWR/SPC` (left thumb) |
| Enter | Tap `RSE/ENT` (right thumb) |
| Backspace | Top-right key |
| Delete | Hold `LWR/SPC` + tap `P` position |
| Tab | Left-hand, second row, far left |
| Escape | Top-left key |
| Caps (via shift) | Hold either `SHFT` key |

### Symbols for Coding

| Symbol | How |
|--------|-----|
| `[ ]` | Right thumb area (default layer) |
| `{ }` | Hold `RSE/ENT` + tap `,` `.` positions |
| `( )` | Hold `RSE/ENT` + tap `O` `P` positions |
| `- _` | Hold `RSE/ENT` + tap `H` or `N` positions |
| `= +` | Hold `RSE/ENT` + tap `J` or `M` positions |
| `\ |` | Hold `RSE/ENT` + tap `/` position (middle or bottom row) |
| `` ` ~ `` | Hold `RSE/ENT` + tap `'` or `SHFT` position |

### Navigation

| Action | How |
|--------|-----|
| Arrow keys | Hold `LWR/SPC` + `N` `M` `,` `.` |
| Home / End | Hold `LWR/SPC` + `H` / `L` |
| Page Up / Down | Hold `LWR/SPC` + `,` / `M` (PgUp/PgDn) |

### Shortcuts (with modifiers)

| Shortcut | How |
|----------|-----|
| Ctrl+C (copy) | Hold `LCTL` + tap `C` |
| Ctrl+V (paste) | Hold `LCTL` + tap `V` |
| Ctrl+Z (undo) | Hold `LCTL` + tap `Z` |
| Ctrl+S (save) | Hold `LCTL` + tap `S` |
| Alt+Tab (switch window) | Hold `ALT` (left thumb) + tap `TAB` |
| Gui/Super (app launcher) | Tap `GUI` (left thumb) |

## Tips for Your First Week

**Day 1–3: Just type.** Stick to Layer 0. Don't worry about speed. Focus on finding the right keys and getting used to the split. It will feel weird — that's normal.

**Day 4–7: Introduce layers.** Start using `LWR/SPC` for arrow keys and `RSE/ENT` for common symbols. The more you use layers, the more natural they'll feel.

**If layer-taps feel off:** Your tapping term is set to 200ms. If you're accidentally triggering layers when you mean to type Space, try increasing it to 250ms. If layers feel sluggish, try lowering it to 150ms. Edit the `tapping-term-ms` value in your keymap file.

**Print this guide** and keep it next to your keyboard for the first couple of weeks. You'll need it less and less as the muscle memory develops.
